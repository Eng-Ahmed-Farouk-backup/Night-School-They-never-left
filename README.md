## Features

- Branching story-telling system (multiple endings)
- Fully custom GUI (dark horror theme) and CLI
- Choice-based gameplay with real consequences
- Horror sound effects
- Psychological horror (not cheap jumpscares, only cheap Jokes)
- max ran who ran YSWSs haha (example on the cheap jokes)

## Architecture
### CLI
- `interactive_consol.py` -> Contains Helper Functions for Desplaying Content in CLI
- `items.py` -> Contains Items Used in game
- `main.py` -> Contains the main menu and opearating to choose between CLI and GUI
### GUI
- `interactive_gui.py` -> contains Helper Functions for Desplaying content in GUI
- `gui.py` -. contains story in GUI and used to run the GUI game
