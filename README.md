# Night School - They never left
this is a GUI RPG game made using python pygame library
![image](assets/main%20bg.png)
## Mechanics
this game uses mechanics like
### 1- scene music
scenes in this game contain music that can be skippable and sometimes not depending on the scene

also these music and sound effects are choosen carefully to make you scared
### 2- choice scenes
these are scenes that contain choices to pick different pathes
### 3- jumpscares
like any horror game, this game contains jumpscares with very scary images and sounds

## Technology used
this game uses technologies like:
pygame for rendering the gui, keyboard library for capturing keyboard key presses for cli
